# Healthcare-Patient-Records-and-Analytics-System-HePRAS-
**Ce système gère les données patients, les horaires des médecins, les rendez-vous, les traitements et la facturation, avec une analytique de base. Il exige une gestion sécurisée, un contrôle d’accès par rôles (RBAC), une intégrité ACID et des requêtes complexes pour les historiques et la facturation.**
